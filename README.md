# AWS Cloud Computing Project

## Overview

This project demonstrates the practical implementation of cloud computing concepts using Amazon Web Services (AWS).

The project involved deploying cloud resources, managing storage, and using AWS command-line tools to demonstrate the interaction between compute and storage services.

## Technologies Used

- Amazon EC2
- Amazon S3
- AWS CloudShell
- AWS CLI

## Key Skills Demonstrated

- Cloud Computing
- AWS Infrastructure
- Cloud Storage
- Infrastructure as a Service (IaaS)
- Command Line Operations

## Project Tasks

- Created an EC2 instance
- Created an S3 bucket
- Uploaded files using AWS CLI
- Verified communication between cloud services
- Compared AWS and Microsoft Azure

## What I Learned

- Managing AWS cloud resources
- Working with EC2 and S3 services
- Using AWS CloudShell and AWS CLI
- Understanding cloud architecture concepts
- Comparing AWS and Azure cloud platforms
